MTTester::
---------------
---------------
Required files:
----------------
1.METTester.jar
2.Class file to test
3.Xml file containing test cases 

How to run a MTTester.jar file from command line?
-----------------------------------------------------
Step 1:
create a 'src' folder inside the path of the METTester.jar file.
Step 2:
Copy the class file under test in the 'src' folder. e.g. MethodCollection2.class
Step 3:
Start command prompt and go the location where METTester.jar file is.
Step 4:
Type "java -cp . -jar METTester.jar" & enter
Step 5:
Type class name to be tested, press enter. E.g. MethodCollection2 
Step 6:
Type the path and file name of the XML and press enter. e.g. C:\METtester\add_values.xml
